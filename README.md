# COREX — Zero-Cost MVP

מטרה: להשיק גרסת MVP ללא ידע וללא תשלום — עם אתר סטטי (GitHub Pages) ואופציונלית וורקר חינמי לענות כ-API מדומה.

## 1) מה יש בתיקיה
- `index.html` — האפליקציה (דשבורד + כרטיס החלטה + פרופיל).
- `data/*.json` — נתונים לדוגמה.
- `cloudflare_worker.js` — שרת קטן אופציונלי כ-API (מדומה).
- הקבצים רצים מקומית ע"י פתיחת `index.html` בדפדפן.

## 2) העלאה ל-GitHub Pages (בחינם)
1. פתח/י חשבון GitHub.
2. צור/י ריפו חדש בשם `corex-mvp` (Public).
3. העלה/י את כל הקבצים שבתיקיה הזו לריפו.
4. ב-Settings → Pages → Source: `Deploy from a branch` ובחר/י את הסניף `main` ואת התיקיה `/ (root)`.
5. קבל/י כתובת אתר חינמית (יעלה מיד/תוך זמן קצר).

## 3) אופציונלי: Cloudflare Worker (בחינם, קל)
1. פתח/י חשבון Cloudflare והתקן/י `wrangler` (מדריך באתר Cloudflare).
2. צור/י Worker חדש, והדבק/י את תוכן `cloudflare_worker.js`.
3. פרסם/י ותקבל/י כתובת `https://xxx.workers.dev/api/decide/optimize`.
4. החלף/י ב-`index.html` את הקריאה לקובץ JSON לקריאת API אמיתי (אם תרצו).

## 4) מה הלאה
- אפשר להחליף בהמשך את ה-API המדומה בשירות אמיתי (FastAPI/Node) — לפי OpenAPI שהכנתי.
- אפשר להרחיב לטפסים וגרפים; ולעבור לנתונים אמיתיים (Google Sheets/Supabase Free).

> שפה חיובית, שקיפות ואחריות: המידע חינוכי וכללי; לפעולות מחייבות — מומלץ מומחה מורשה לפי הדין המקומי.
